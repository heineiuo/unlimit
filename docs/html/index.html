Web操作界面

所有的操作都不需要编写配置脚本, 全部通过web界面操作完成. 你可以在线配置反向代理, 文件代理, vHost配置, 甚至定义html和json等格式的返回数据.

自定义域名和路由

当域名指向你的服务器IP后,你就可以通过web界面设置该域名的返回结果. 你可以给域名 添加路由, 对每一个路由你可以指定多种返回结果,包括html, json, 端口代理, 文件代理, 甚至是代理http请求到指定的url, 并返回结果. 你还可以添加诸如允许跨域这样的返回头.

Node.js程序管理

可以通过web管理界面, 你可以监控正在运行的Node.js程序的运行状态, 通过指定入口文件地址 启动新的Node.js程序.
