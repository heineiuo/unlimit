# Chat

### 1： install

#### 依赖：
* node.js: ^4.0.0
* npm: ^3.0.0
* babel-cli 
* chrome

命令行：

```
$ yarn
```

### 2: 进入开发模式

注意，安装完后先根据3编译一次， 再进入开发模式

命令行：

```
$ npm start
```

chrome打开 `http://127.0.0.1:8083`

入口文件是 `src/index.js`

### 3: 编译
命令行

```
$ npm run build
```

生成文件在 `build/public/chat` 目录
